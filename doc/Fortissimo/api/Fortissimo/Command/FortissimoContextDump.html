Class FortissimoContextDump

Description
Description | Vars | Methods (details)

This command dumps the contents of the context to standard out.

It performs this operation by running http://www.php.net/var_dump on the contents of the FortissimoExecutionContext.

This is useful (occasionally) for debugging.
