Class FortissimoArrayInjectionLogger

Description
Description | Descendents | Vars (details) | Methods (details)

The FAIL logger maintains an array of messages to be retrieved later.

Log entries can be injected into the output by retrieving a list of log messages with getMessages(), and then displaying them, or by simply calling printMessages().
