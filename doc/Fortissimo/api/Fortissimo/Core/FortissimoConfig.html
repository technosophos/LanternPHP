Class FortissimoConfig

Description
Description | Vars (details) | Methods (details)

Stores information about Fortissimo commands.

This is used when bootstrapping to map a request to a series of commands. Note that this does not provide an object represenation of the configuration file. Instead, it interprets the configuration file, and assembles the information as the application needs it. To get directly at the configuration information, use getConfig().

  • see: Fortissimo

Create a command instance.

  • return: An array with the following keys:
